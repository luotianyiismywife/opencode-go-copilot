# Future Plans

## 1. 多 Go 工作区支持
扩展为自动发现并收集所有有 Go 订阅的工作区的 API Key，而非只取第一个。每个工作区管理独立的 Key，用户可选择或自动切换。

## 2. 用量监控
将用量信息显示到 VS Code 状态栏：
- 外部只显示 **5小时滚动用量**
- 鼠标悬停时显示 **5小时 / 周 / 月** 三个维度
- 每 5 分钟自动从网页 SSR 数据获取一次
- 切换工作区/API Key 时也自动拉取新用量

## 3. 429 自动切换
- 收到 429（额度不足/限流）时自动换到下一个可用的 API Key
- 所有 Key 都用完后提示用户切换到免费模型继续使用
